Conceitos de Informática

1.1 Componentes Internos

1.1.1 CPU

A CPU (Unidade Central de Processamento) é o cérebro do computador, responsável por executar operações e processar informações.

1.1.2 Armazenamento

Os dispositivos de armazenamento permitem salvar dados e programas permanentemente, como HDs e SSDs.

1.1.3 Memória

A memória RAM é responsável por armazenar temporariamente dados e programas em execução.

1.1.4 Fonte de alimentação

A fonte de alimentação converte a energia elétrica da tomada para um formato adequado aos componentes do computador.

1.2 Componentes Externos

1.2.1 Teclado

O teclado é um dispositivo de entrada utilizado para inserir informações no computador.

1.2.2 Monitor de vídeo

O monitor exibe informações visuais processadas pelo computador.

1.2.3 Mouse

O mouse é um dispositivo de entrada que permite movimentar o cursor e interagir com a interface gráfica.

2. Grandezas Elétricas

2.1 Potência elétrica

A potência elétrica representa a quantidade de energia consumida ou fornecida por um dispositivo em um determinado tempo.

2.2 Tensão elétrica

A tensão elétrica é a diferença de potencial entre dois pontos de um circuito elétrico.

2.3 Corrente elétrica

A corrente elétrica é o fluxo de elétrons através de um condutor.

3. Unidades de Medida Utilizadas na Informática

3.1 Capacidade de Armazenamento

3.1.1 bit

O bit é a menor unidade de informação da computação e pode representar 0 ou 1.

3.1.2 Byte

O byte é composto por 8 bits e é a unidade básica de armazenamento digital.

3.1.3 Múltiplos

Os múltiplos do byte incluem kilobyte (KB), megabyte (MB), gigabyte (GB), entre outros.

3.2 Frequência

3.2.1 Hertz

O hertz (Hz) é a unidade de medida da frequência, representando ciclos por segundo.

3.2.2 Múltiplos

Os múltiplos do hertz incluem quilohertz (kHz), megahertz (MHz) e gigahertz (GHz).

3.4 Taxa de Transferência

3.4.1 Tráfego no canal

O tráfego no canal refere-se à quantidade de dados transmitidos em um determinado período.

3.4.2 Byte/seg (Bps)

O byte por segundo (Bps) é uma unidade de medida da taxa de transferência de dados.
